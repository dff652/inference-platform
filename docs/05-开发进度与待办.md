## 三、已知技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| SQLite 并发写入限制 | Celery worker 并发写 DB 可能冲突 | 一期单 worker；二期切 PostgreSQL |
| Qwen3.5-27B 显存不足 | 2×22GB 不够 4-bit 加载 | 代码已支持多卡和 4-bit，换 A100/4090 后可用 |
| ChatTS-8B 4-bit 质量 | force 模式下量化可能降低检测精度 | 后续可在大显存设备上使用 bf16 全精度 |
| 旧项目 run.py 输出格式不稳定 | 不同方法的 JSON 结构可能有差异 | 结果解析已做容错处理 |
