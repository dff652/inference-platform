## 目录结构


## API 接口清单

### 推理任务 /api/v1/inference/tasks

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 创建任务 |
| GET | / | 任务列表（支持 status/submitter/algorithm 筛选） |
| GET | /stats | 各状态任务统计 |
| GET | /{id} | 任务详情 |
| PUT | /{id} | 更新任务 |
| POST | /{id}/submit | 提交执行 |
| POST | /{id}/cancel | 取消任务 |
| POST | /{id}/retry | 重试任务 |
| GET | /{id}/results | 推理结果详情（含 segments） |
| GET | /{id}/logs | 执行日志 |
| GET | /{id}/results/{rid}/chart-data | D3 图表数据 |

### 同步预测 /api/v1/predict

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 同步预测（直接返回结果，不入 Celery） |

### 模型中心 /api/v1/models

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 注册模型 |
| GET | / | 模型列表（支持 family/status 筛选） |
| GET | /{id} | 模型详情 |
| PUT | /{id} | 更新模型 |
| POST | /{id}/activate | 激活模型 |
| POST | /{id}/archive | 归档模型 |
| GET | /vllm/status | GPU 服务健康状态 |

### 配置管理 /api/v1/inference/configs

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /algorithms | 支持的算法列表（含 resource 标记） |
| GET | /chatts-prompts | ChatTS Prompt 模板列表 |
| POST | / | 创建配置模板 |
| GET | / | 配置列表 |
| PUT | /{id} | 更新配置 |

### 数据源 /api/v1/data-sources

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 创建数据源 |
| GET | / | 数据源列表 |
| GET | /{id} | 数据源详情 |
| PUT | /{id} | 更新数据源 |
| POST | /{id}/test | 连通性测试 |

### 文件上传 /api/v1/uploads

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 上传 CSV/TXT/XLSX 文件 |

## 任务状态机

```
draft → pending → queued → running → completed
                                   → failed    → pending (retry)
                                   → cancelled
                                   → timeout   → pending (retry)

可取消状态：draft, pending, queued, running
可重试状态：failed, timeout
```

## 执行调度链

```
API submit → Celery task dispatch
  ├── CPU 方法 (ensemble/wavelet/iforest/adtk_hbos/stl_wavelet)
  │   → _run_direct() → dispatcher.run() → 算法模块直接调用 → lb_eval 打分
  │
  ├── GPU 方法 (chatts/qwen)
  │   → _run_vllm() → dispatcher.run_vllm() → HTTP POST 到 GPU 服务
  │     ├── chatts_serve.py (:8001) → transformers 推理 (chatts conda env)
  │     └── vLLM serve     (:8002) → vLLM 0.11.0 推理 (qwen-vllm011-clean conda env)
  │   → 解析模型输出 → lb_eval 打分
  │
  └── 回退 (其他方法)
      → _run_subprocess() → CLIExecutorAdapter → subprocess 调旧 run.py

所有路径 → 写结果文件 (CSV + metrics.json + segments.json)
         → 更新 DB (status + result_summary + InferenceResultIndex)
         → 写执行日志 (execution.log)
```
