## API 接口清单

### 推理任务 /api/v1/inference/tasks

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 创建任务 |
| GET | / | 任务列表（支持 status/submitter/algorithm 筛选） |
| GET | /stats | 各状态任务统计 |
| GET | /{id} | 任务详情 |
| PUT | /{id} | 更新任务 |
| POST | /{id}/submit | 提交执行 |
| POST | /{id}/cancel | 取消任务 |
| POST | /{id}/retry | 重试任务 |

### 模型中心 /api/v1/models

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 注册模型 |
| GET | / | 模型列表（支持 family/status 筛选） |
| GET | /{id} | 模型详情 |
| PUT | /{id} | 更新模型 |
| POST | /{id}/activate | 激活模型 |
| POST | /{id}/archive | 归档模型 |

### 配置管理 /api/v1/inference/configs

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /algorithms | 支持的算法列表 |
| POST | / | 创建配置模板 |
| GET | / | 配置列表 |
| PUT | /{id} | 更新配置 |

### 数据源 /api/v1/data-sources

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 创建数据源 |
| GET | / | 数据源列表 |
| GET | /{id} | 数据源详情 |
| PUT | /{id} | 更新数据源 |
| POST | /{id}/test | 连通性测试 |

## 任务状态机

```
draft → pending → queued → running → completed
                                   → failed    → pending (retry)
                                   → cancelled
                                   → timeout   → pending (retry)

可取消状态：draft, pending, queued, running
可重试状态：failed, timeout
```

## 执行适配器调用链

```
API submit → Celery task dispatch → CLIExecutorAdapter
  → subprocess: python run.py --method chatts --input file.csv --task-id 123
  → 结果收集：解析 output_dir 下的 JSON 文件
  → 状态回写：completed / failed + error_message
  → 日志保存：execution.log
```
